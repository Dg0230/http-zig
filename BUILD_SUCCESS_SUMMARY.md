## 📋 使用说明

### 基本启动
```bash
# 开发模式
zig build run

# 生产模式构建
zig build -Doptimize=ReleaseFast

# 运行测试
zig build test

# 性能测试
zig build test-perf
```

### 配置选项
- **端口**: 默认 8080 (可配置)
- **地址**: 默认 127.0.0.1 (可配置)
- **最大连接数**: 默认 1000 (可配置)
- **缓冲区大小**: 默认 4KB (可配置)

### API 端点
- `GET /` - 主页
- `GET /api/users` - 用户列表
- `POST /api/users` - 创建用户
- `GET /api/users/:id` - 获取用户
- `PUT /api/users/:id` - 更新用户
- `DELETE /api/users/:id` - 删除用户
